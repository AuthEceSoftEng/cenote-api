# smack-bdms-web

>BDMS Management Dashboard

## Quick Start

* Install dependencies:

```bash
 $ yarn
 ```

* Start mongo deamon:

```bash
 $ mongod
 ```

### Development

```bash
$ yarn dev
```

### Production

```bash
$ yarn prod
```

### Docker

```bash
$ docker-compose build
$ docker-compose up
```

#### Other Commands

```bash
yarn build
yarn test
yarn test:verbose
yarn lint
yarn lint:fix
```
